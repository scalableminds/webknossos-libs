# Claude Code Instructions

## Testing

Run tests with `uv run test.py` (not `python -m pytest`).

## Formatting

After code changes, run `./format.sh` to format the code.

## Typechecking

After code changes, run `./typecheck.sh` to typecheck the code.

## Linting

After code changes, run `./lint.sh` to lint the code.

## Documentation

For CLI commands, the docs are in `../docs/src/cli/`.
New pages need to be registered in `../docs/mkdocs.yml`.
